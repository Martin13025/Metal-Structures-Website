# Metal Site

Веб-приложение для компании по изготовлению металлоконструкций.  
Frontend: **React**  
Backend: **Node.js (Express)**  
База данных: **MySQL**

---

## 1. Требования

Перед запуском убедитесь, что установлено:

- Node.js **18+**
- npm
- MySQL (удобно через **XAMPP**)
- Git (по желанию)

---

## 2. Установка зависимостей

### Frontend

В корне проекта:

```bash
npm install

Backend

cd server
npm install
```
## После
```bash
cd ./metal-site
npm start
```
```bash
cd ./metal-site
cd ./server
node server.js
